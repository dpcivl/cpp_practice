# 1) i2c-tools

Linux 시스템에서 I2C 버스와 연결된 장치들을 다루기 위한 유틸리티 모음.

## 1.1> i2c-tools 주요 구성 요소
- i2cdetect : I2C 버스를 스캔하는 용도
- i2cget : I2C 버스에 연결된 장치로부터 바이트나 워드 단위의 데이터를 읽음
- i2cset : I2C 버스에 연결된 장치에 바이트나 워드 단위의 데이터를 씀
- i2cdump : I2C 버스에 연결된 장치의 레지스터 내용을 덤프함.

여기까지 찾아봤는데 i2c를 제어하는 건 맞는데 이건 커맨드 상에서 제어하는 거라고 한다.
