TCP/IP 통신을 사용해서 데이터를 주고 받아보았다. 
프로토콜을 만들어서 상태 감시와 다른 장비의 정보를 받아 데이터를 표출해보는 것을 목표로 한다. 
# 1) 요구사항
- 장비(서버)에게 데이터 요청
- 받은 데이터에서 필요한 부분만 추출
- 추출한 데이터를 프로토콜 구성하는 데 사용
- 나머지 데이터들도 같은 방식으로 세팅

# 2) 구현
## 2.1> 장비에게 데이터 요청
프로토콜을 만드는데, 각 항목 별로 알기 쉽게 표현해야 한다. (KMA)
`makePacket()`이라는 메소드를 사용해서 보낼 데이터를 준비한다. 
준비한 데이터를 59초에 보낸다. ➡️ 시간 불러오기 필요함.
