소켓을 만든 뒤, 데이터를 주지는 않고 일단 받는 작업만 해볼 계획이다. 

# 1) TCP/IP 통신 개념
## 1.1> TCP 서버 소켓 생성 및 바인딩
1. 소켓 생성: `socket()` 함수를 사용하여 소켓을 생성한다. 
2. 소켓 옵션 설정: `setsockopt()` 함수를 사용하여 소켓 옵션을 설정할 수 있다. 
3. 주소 바인딩: `bind()` 함수를 사용하여 서버의 주소(포트와 IP)를 소켓에 바인딩한다. 
4. 리스닝 시작: `listen()` 함수를 사용하여 들어오는 연결에 대해 소켓을 리스닝 상태로 만든다. 
5. 연결 수락: `accept()` 함수를 사용하여 클라이언트 연결을 수락한다. 

## 1.2> TCP 클라이언트 소켓 생성 및 서버 연결
1. 소켓 생성: `socket()` 함수를 사용하여 소켓을 생성한다. 
2. 서버 연결: `connect()` 함수를 사용하여 서버에 연결한다.
